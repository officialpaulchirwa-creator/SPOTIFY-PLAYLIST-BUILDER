Setup Guide

You need a free Spotify Developer app and an Anthropic API key. Takes about 3 minutes.

  1. Go to developer.spotify.com/dashboard, log in, create a new app, and set the Redirect URI to http://localhost.
  2. Copy your Client ID from the dashboard and paste it in the sidebar.
  3. Click Connect Spotify — authorize, then copy the URL you land on and paste it when prompted. Token is extracted automatically.
  4. Get your Anthropic key at console.anthropic.com.
